## 1. Headings

Markdown supports six levels of headings, using `#` symbols.

```markdown
# H1 - Main Heading
## H2 - Subheading
### H3 - Sub-subheading
#### H4 - Sub-sub-subheading
##### H5 - Fifth level heading
###### H6 - Sixth level heading
```

## 2. Text Formatting

### Bold and Italic

You can format text as **bold** or *italic*.

```markdown
**This is bold text**
*This is italic text*
```

### Bold and Italic Together

You can also combine both:

```markdown
***This is bold and italic text***
```

## 3. Lists

Markdown supports ordered and unordered lists.

### Unordered List

```markdown
- Item 1
- Item 2
- Item 3
```

### Ordered List

```markdown
1. First item
2. Second item
3. Third item
```

## 4. Links

To create a hyperlink, use the following format:

```markdown
[GitHub](https://github.com)
```

## 5. Images

To add an image, use this syntax:

```markdown
![Alt text](https://example.com/image.png)
```

## 6. Code

### Inline Code

For inline code, use backticks:

```markdown
Here is some `inline code`.
```

### Code Blocks

For code blocks, use triple backticks:

```markdown
```
console.log('Hello World!');
```
```

## 7. Blockquotes

Blockquotes are used for quoting text or content:

```markdown
> This is a blockquote.
```

## 8. Tables

You can create tables using pipes (`|`) and dashes (`-`):

```markdown
| Header 1 | Header 2 |
|----------|----------|
| Row 1, Col 1 | Row 1, Col 2 |
| Row 2, Col 1 | Row 2, Col 2 |
```
